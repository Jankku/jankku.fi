Projects

Eino

Book and movie tracker, Full-stack project. Express.JS TypeScript backend with PostgreSQL database, React.JS frontend and Kotlin Android application. In addition to JWT authentication and book/movie CRUD actions, both clients have search feature and profile page for statistics. Backend testing is done with Chai and Mocha. Deployed with Docker Compose.

Yle RSS Reader

Yle regional RSS feed reader, React Native/Expo. Location based region fetching with manual option. Features include saving, sharing and reading articles. Article reader or WebView for reading. Articles are saved in a Realm database. Unit and snapshot tests with Jest and Testing Library. Light/dark themes.

AlphaWall

Wallpaper Android application. Kotlin, MVVM, Retrofit, Glide, Moshi, Jetpack libraries. AlphaCoders API used for content. Sort wallpapers by views, rating, favorites and newest. Set pictures as wallpaper or download it to your device. Search with keywords.

Notes

Simple note-taking Android application written in Kotlin. Features include press back to save, open keyboard when opening note, export/import notes, and language switcher (fin/eng). Has material design and light/dark themes. This is my first Android app I have made.
